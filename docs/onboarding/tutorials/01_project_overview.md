# 教程 01: 项目概览

## 学习目标

理解飞控系统的基本架构和本项目的组织方式。

## 飞控系统组成

```
传感器 → 状态估计 → 控制器 → 执行器
  ↑                           ↓
  └───────── 反馈 ←──────────┘
```

### 核心模块

1. **传感器融合**: IMU、气压计、GPS等数据融合
2. **状态估计**: 姿态、位置、速度估计
3. **控制器**: 姿态环 + 位置环级联控制
4. **执行器**: 电机PWM输出

## 项目两大模块

### theory/ - 理论研发
- 在这里进行数学建模和算法设计
- 使用 MATLAB/Python 进行仿真验证
- 输出经过验证的算法文档

### implementation/ - 工程实装
- 将算法转化为嵌入式代码
- 处理实时性、硬件限制等工程问题
- 进行实机测试
